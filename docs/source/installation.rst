Installation
============

The package can be installed from PyPi:

.. code-block:: shell

    pip install tmplot


Or directly from this repository:

.. code-block:: shell

    pip install git+https://github.com/maximtrp/tmplot.git


Dependencies
~~~~~~~~~~~~

* ``numpy``
* ``scipy``
* ``scikit-learn``
* ``pandas``
* ``altair``
* ``ipywidgets``
* ``tomotopy``, ``gensim``, and ``bitermplus`` (for topic modeling)
